Add theme validation CLI tool

New package `@grasdouble/lufa_design-system-cli` for validating custom themes:
- Completeness check: ensures all 453 tokens are defined
- Contrast check: validates WCAG AA contrast ratios
- Format check: validates CSS value formats
- Theme template generator via --template flag
- Colored CLI output with verbose mode
