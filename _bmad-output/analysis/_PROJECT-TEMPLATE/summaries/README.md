---

### Summary vs. Other Document Types

**Phase Summary (summaries/):**

- **What:** Completion record
- **When:** After phase is done
- **Content:** What actually happened
- **Purpose:** Historical record

**Analysis/Study (studies/):**

- **What:** Deep dive analysis
- **When:** During or after work
- **Content:** Why/how decisions were made
- **Purpose:** Reference material

**Work in Progress (current/):**

- **What:** Active work document
- **When:** During phase
- **Content:** Ongoing notes and drafts
- **Purpose:** Track active work
