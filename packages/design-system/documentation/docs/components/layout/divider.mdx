# Divider

A lightweight separator for grouping content. Supports horizontal and vertical orientations, optional labels, solid or dashed styles, and built-in spacing.

## Overview

Divider creates a visual break between sections of content. It can render a centered label, align the label to start or end, switch to a vertical orientation for side-by-side layouts, and choose between solid or dashed rules. Use Divider to clarify hierarchy and transitions in pages, cards, and panels without adding heavy visual weight.


## Props

| Prop          | Type                             | Default        | Description                                                     |
| ------------- | -------------------------------- | -------------- | --------------------------------------------------------------- |
| `label`       | `ReactNode`                      | -              | Optional label displayed between the lines (horizontal only)    |
| `orientation` | `'horizontal' \| 'vertical'`     | `'horizontal'` | Direction of the divider                                        |
| `variant`     | `'solid' \| 'dashed'`            | `'solid'`      | Line style                                                      |
| `align`       | `'start' \| 'center' \| 'end'`   | `'center'`     | Label alignment for horizontal dividers                         |
| `spacing`     | `'none' \| 'sm' \| 'md' \| 'lg'` | `'md'`         | Vertical margin applied around the divider                      |
| `length`      | `number \| string`               | -              | Constrain width (horizontal) or height (vertical). Accepts px/% |
| `className`   | `string`                         | -              | Additional CSS classes                                          |

## Use Cases

### 1. Section Headlines

Break long forms or pages into smaller, scannable sections with labeled dividers.

### 2. Split Layouts

Use vertical dividers to separate columns inside dashboards, cards, or toolbars.

## Styling

Divider uses semantic border tokens for subtle separation and inherits text color for labels.

- **Base styles**: horizontal or vertical rule with `border-border-light`, optional label with medium font weight.
- **Line styles**: `variant="solid"` (default) or `variant="dashed"` apply corresponding border styles.
- **Spacing**: `spacing` adds vertical margin around the divider (`none`, `sm`, `md`, `lg`).
- **Customization**: Extend with `className` or wrap in your own container to control width/height via `length`.

## Accessibility

- Renders `role="separator"` with `aria-orientation` to convey direction to assistive tech.
- Labels are plain text; include meaningful copy when using `label`.
- Not keyboard focusable by default; keep interactive controls outside the divider itself.
- Ensure sufficient color contrast between the rule and its background.

## Best Practices

### Do ✅

- Use labels to clarify transitions between major sections.
- Constrain `length` when the divider should not span the full container.
- Choose `variant="dashed"` for lighter separation in dense layouts.

### Don't ❌

- Rely on Divider alone to communicate critical state (pair with text/icons).
- Overuse vertical dividers in narrow layouts where they add visual clutter.
