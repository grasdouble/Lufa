# Card

A flexible container component for grouping related content. Supports a title/subtitle header, optional footer actions, multiple visual variants, and padding sizes.

## Overview

Use `Card` to group related information and actions into a clear surface.

- Header support via `title` / `subtitle`
- Footer slot via `footer` (actions, meta, etc.)
- Visual styles via `variant` and spacing via `padding`
- Interactive affordance via `hoverable` + `onClick`

## Installation

```tsx
import { Card } from "@grasdouble/lufa_design-system";
```

## Basic Usage

```tsx
<Card title="Title" subtitle="Optional subtitle">
  Content goes here.
</Card>
```

## Props

| Prop       | Type                                          | Default     | Description                          |
| ---------- | --------------------------------------------- | ----------- | ------------------------------------ |
| `children` | `ReactNode`                                   | -           | Card content                         |
| `title`    | `string`                                      | -           | Card title (header)                  |
| `subtitle` | `string`                                      | -           | Card subtitle/description (header)   |
| `footer`   | `ReactNode`                                   | -           | Footer slot (actions, meta, etc.)    |
| `variant`  | `'default' \| 'elevated' \| 'outlined' \| 'filled'` | `'default'` | Visual variant                       |
| `padding`  | `'none' \| 'small' \| 'medium' \| 'large'`    | `'medium'`  | Padding size                         |
| `hoverable`| `boolean`                                     | `false`     | Add hover affordance                 |
| `onClick`  | `() => void`                                  | -           | Click handler (makes card clickable) |
| `className`| `string`                                      | -           | Additional CSS classes               |
