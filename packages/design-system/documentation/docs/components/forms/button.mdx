# Button

A versatile button component for actions and navigation. Supports variants, colors, sizes, icons, and loading/disabled states.

## Overview

Use `Button` for primary actions, secondary actions, and links that should look like buttons.

- Style via `variant` and `color`
- Size via `size` and layout via `fullWidth`
- Async states via `loading` and `disabled`
- Icons via `startIcon` / `endIcon`

## Installation

```tsx
import { Button } from "@grasdouble/lufa_design-system";
```

## Basic Usage

```tsx
<Button>Action</Button>

<Button variant="outlined">Secondary action</Button>
```

## Props

| Prop        | Type                                                     | Default    | Description                                   |
| ----------- | -------------------------------------------------------- | ---------- | --------------------------------------------- |
| `children`  | `ReactNode`                                              | -          | Button content (text, icons, etc.)            |
| `label`     | `string`                                                 | -          | Deprecated (use `children`)                   |
| `variant`   | `'solid' \| 'outlined' \| 'text' \| 'ghost' \| 'link'`    | `'solid'`  | Visual style variant                          |
| `color`     | `'primary' \| 'secondary' \| 'success' \| 'warning' \| 'danger'` | `'primary'` | Color scheme                                  |
| `size`      | `'small' \| 'medium' \| 'large'`                         | `'medium'` | Size                                          |
| `fullWidth` | `boolean`                                                | `false`    | Make button full width                        |
| `loading`   | `boolean`                                                | `false`    | Show loading state (also disables the button) |
| `disabled`  | `boolean`                                                | `false`    | Disable interactions                          |
| `startIcon` | `ReactNode`                                              | -          | Icon before content                           |
| `endIcon`   | `ReactNode`                                              | -          | Icon after content                            |
| `onClick`   | `(event: React.MouseEvent<HTMLButtonElement>) => void`   | -          | Click handler                                 |
| `type`      | `'button' \| 'submit' \| 'reset'`                        | `'button'` | Button type attribute                         |
