# Interactive Playground

Try out Lufa Design System components right here in your browser! Edit the code below to see changes in real-time.

## Basic Button Example

Click "Edit" to modify the code and see your changes instantly.

```tsx live
import * as React from 'react';
import { Button, Stack, Text } from '@grasdouble/lufa_design-system';

function ButtonDemo() {
  const [count, setCount] = React.useState(0);

  return (
    <Stack direction="vertical" spacing="comfortable">
      <Button type="solid" variant="primary" onClick={() => setCount(count + 1)}>
        Clicked {count} times
      </Button>
      <Text variant="body">Counter: {count}</Text>
    </Stack>
  );
}
```

## Button Variants

Explore different button styles:

```tsx live
import { Button, Stack } from '@grasdouble/lufa_design-system';

function ButtonVariants() {
  return (
    <Stack direction="horizontal" spacing="default" wrap={true}>
      <Button type="solid" variant="primary">Primary</Button>
      <Button type="outline" variant="primary">Outline</Button>
      <Button type="ghost" variant="primary">Ghost</Button>
      <Button type="solid" variant="secondary">Secondary</Button>
    </Stack>
  );
}
```

## Stack Layout

Create flexible layouts with the Stack component:

```tsx live
import { Stack, Text } from '@grasdouble/lufa_design-system';

function StackDemo() {
  return (
    <Stack direction="vertical" spacing="comfortable">
      <Stack direction="horizontal" spacing="default">
        <div style={{ padding: '1rem', background: '#e0f2fe', borderRadius: '8px', minWidth: '100px' }}>Box 1</div>
        <div style={{ padding: '1rem', background: '#ddd6fe', borderRadius: '8px', minWidth: '100px' }}>Box 2</div>
        <div style={{ padding: '1rem', background: '#fce7f3', borderRadius: '8px', minWidth: '100px' }}>Box 3</div>
      </Stack>

      <Text variant="body">Items arranged horizontally with default spacing</Text>
    </Stack>
  );
}
```

## Interactive Card

Build a complete card component:

```tsx live
import * as React from 'react';
import { Box, Button, Stack, Text } from '@grasdouble/lufa_design-system';

function CardDemo() {
  const [liked, setLiked] = React.useState(false);

  return (
    <Box
      padding="comfortable"
      background="surface"
      borderRadius="base"
      style={{ boxShadow: 'var(--lufa-semantic-ui-shadow-medium)' }}
    >
      <Stack direction="vertical" spacing="compact">
        <Text as="h3" variant="h3">Interactive Card</Text>
        <Text variant="body">
          This is a card component with interactive elements. Click the button below to toggle the like state.
        </Text>
        <Button
          type={liked ? 'solid' : 'outline'}
          variant="primary"
          onClick={() => setLiked(!liked)}
        >
          {liked ? 'Liked' : 'Like'}
        </Button>
      </Stack>
    </Box>
  );
}
```

## Tips for Using the Playground

- All Lufa Design System components are available in the live editor
- Changes are reflected instantly as you type
- Use `React.useState` for interactive examples
- Try combining multiple components to build complex UIs
- Check the [Component Overview](/docs/components/overview) for the full component API

## Available Components

The following components are available in the playground:

- **Layout**: `Stack`, `Container`, `Flex`, `Grid`, `Divider`, `Center`, `Space`, `AspectRatio`, `Placeholder`, `Layout`
- **Text**: `Text` (with variants: h1-h6, body-large, body, body-small, caption, label)
- **Display**: `Card`, `Avatar`, `AvatarGroup`, `Badge`
- **Forms**: `Button`, `Input`
- **Navigation**: `Anchor`, `Link`, `Breadcrumb`, `Tabs`, `Menu`, `Pagination`, `Steps`
- **Feedback**: `Alert`, `Spinner`
- **Overlay**: `Modal`
- **Patterns**: `Testimonial`

See the [full component list](/docs/components/overview) for detailed documentation on each component.
