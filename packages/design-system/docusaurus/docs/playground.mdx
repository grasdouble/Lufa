# Interactive Playground

Try out Lufa Design System components right here in your browser! Edit the code below to see changes in real-time.

## Basic Button Example

Click "Edit" to modify the code and see your changes instantly.

```tsx live
function ButtonDemo() {
  const [count, setCount] = React.useState(0);

  return (
    <Stack direction="vertical" gap="normal">
      <Button variant="solid" color="primary" onClick={() => setCount(count + 1)}>
        Clicked {count} times
      </Button>
      <Typography variant="body">Counter: {count}</Typography>
    </Stack>
  );
}
```

## Stack Layout

Create flexible layouts with the Stack component:

```tsx live
function StackDemo() {
  return (
    <Stack direction="vertical" gap="spacious">
      <Stack direction="horizontal" gap="normal">
        <div style={{ padding: '1rem', background: '#e0f2fe', borderRadius: '8px', minWidth: '100px' }}>Box 1</div>
        <div style={{ padding: '1rem', background: '#ddd6fe', borderRadius: '8px', minWidth: '100px' }}>Box 2</div>
        <div style={{ padding: '1rem', background: '#fce7f3', borderRadius: '8px', minWidth: '100px' }}>Box 3</div>
      </Stack>

      <Divider />

      <Typography variant="body">Items arranged horizontally with normal spacing</Typography>
    </Stack>
  );
}
```

## Tips for Using the Playground

- All Lufa Design System components are available in the live editor
- Changes are reflected instantly as you type
- Use `React.useState` for interactive examples
- Try combining multiple components to build complex UIs
- Check the [Component Overview](/docs/components/overview) for the full component API

## Available Components

The following components are available in the playground:

- **Layout**: `Stack`, `Container`, `Flex`, `Grid`, `Divider`, `Center`, `Space`, `AspectRatio`, `Placeholder`, `Layout`
- **Typography**: `Typography` (with variants: h1-h6, body, bodyLarge, bodySmall, caption, label)
- **Display**: `Card`, `Avatar`, `AvatarGroup`, `Badge`
- **Forms**: `Button`, `Input`
- **Navigation**: `Anchor`, `Link`, `Breadcrumb`, `Tabs`, `Menu`, `Pagination`, `Steps`
- **Feedback**: `Alert`, `Spinner`
- **Overlay**: `Modal`
- **Patterns**: `Testimonial`

See the [full component list](/docs/components/overview) for detailed documentation on each component.
