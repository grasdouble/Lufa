# Interactive Playground

Try out Lufa Design System components right here in your browser! Edit the code below to see changes in real-time.

## Basic Button Example

Click "Edit" to modify the code and see your changes instantly.

```tsx live noInline
const ButtonDemo = () => {
  const [count, setCount] = React.useState(0);

  return (
    <Stack direction="vertical" spacing="comfortable">
      <Button onClick={() => setCount(count + 1)}>Clicked {count} times</Button>
      <Text>Counter: {count}</Text>
    </Stack>
  );
};

render(<ButtonDemo />);
```

## Stack Layout

Create flexible layouts with the Stack component:

```tsx live noInline
const StackDemo = () => {
  return (
    <Stack direction="vertical" spacing="comfortable">
      <Stack direction="horizontal" spacing="default">
        <div
          style={{
            padding: '1rem',
            background: 'var(--lufa-token-color-background-secondary)',
            borderRadius: '8px',
            minWidth: '100px',
          }}
        >
          Box 1
        </div>
        <div
          style={{
            padding: '1rem',
            background: 'var(--lufa-token-color-background-secondary)',
            borderRadius: '8px',
            minWidth: '100px',
          }}
        >
          Box 2
        </div>
        <div
          style={{
            padding: '1rem',
            background: 'var(--lufa-token-color-background-secondary)',
            borderRadius: '8px',
            minWidth: '100px',
          }}
        >
          Box 3
        </div>
      </Stack>

      <Text>Items arranged horizontally with default spacing</Text>
    </Stack>
  );
};

render(<StackDemo />);
```

## Interactive Card

Build a complete card component:

```tsx live noInline
const CardDemo = () => {
  const [liked, setLiked] = React.useState(false);

  return (
    <Box
      padding="comfortable"
      background="surface"
      borderRadius="base"
      style={{ boxShadow: 'var(--lufa-semantic-ui-shadow-medium)' }}
    >
      <Stack direction="vertical" spacing="compact">
        <Text as="h3" variant="h3">
          Interactive Card
        </Text>
        <Text>
          This is a card component with interactive elements. Click the button below to toggle the like state.
        </Text>
        <Button type={liked ? 'solid' : 'outline'} variant="primary" onClick={() => setLiked(!liked)}>
          {liked ? 'Liked' : 'Like'}
        </Button>
      </Stack>
    </Box>
  );
};

render(<CardDemo />);
```

## Tips for Using the Playground

- All Lufa Design System components are available in the live editor
- Changes are reflected instantly as you type
- Use `React.useState` for interactive examples
- Try combining multiple components to build complex UIs
- Check the [Component Overview](/docs/components/overview) for the full component API

## Available Components

The following components are available in the playground:

- **Layout**: `Stack`, `Container`, `Flex`, `Grid`, `Divider`, `Center`, `Space`, `AspectRatio`, `Placeholder`, `Layout`
- **Text**: `Text` (with variants: h1-h6, body-large, body, body-small, caption, label)
- **Display**: `Card`, `Avatar`, `AvatarGroup`, `Badge`
- **Forms**: `Button`, `Input`
- **Navigation**: `Anchor`, `Link`, `Breadcrumb`, `Tabs`, `Menu`, `Pagination`, `Steps`
- **Feedback**: `Alert`, `Spinner`
- **Overlay**: `Modal`
- **Patterns**: `Testimonial`

See the [full component list](/docs/components/overview) for detailed documentation on each component.
